Vena

Python - Discord Bot

Vena was a bot developed for the Discord platform using Python. The bot served as a tool to manage characters and items in an online Dungeons and Dragons inspired game. The functionalities of the bot involved creating new characters, with the use of templates or with full randomization, or managing character’s inventory with the use of simple commands accessible to all users. On top of that, bot has stored a list of all users and allowed them to log into the characters they created.
Above screenshot lists all the information for an user created character. Aside from storing stats, spells and the inventory, it also informs the user that nobody is currently logged in as the character.

The user base for which the bot was developed were Polish speakers. Therefore the UI of the bot is written fully in Polish, to deliver the best user experience. Sadly, due to the issues with Repl hosting, as well as several updates to the Discord API, the bot is no longer online.
