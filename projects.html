Portfolio Website

Languages: PHP, HTML, CSS, JavaScript

Software: Visual Studio Code

Please note! This version of the website that is currently hosted on Github uses only static code, as Github Pages does not support PHP.

How it started

I have made this website to act as a portfolio and support my CV. The project I have done during the “Introduction to web development” module at the university has served as a base for making. The entire website uses dynamically generated content. For example, this section is loaded onto the “Projects” page from an .xml file that holds the HTML for this article.

Object Oriented Approach

The pages that you can access above, that display the details of my games, are built using custom “game” objects that simply load all the data to generate the information on the page. While using such methods for a page with so little content may seem excessive, being able to easily maintain and expand it is important to me.

Development and the future

I have started working on this website on 06.10.2022, and intend to expand it further over time. The initial deadline was to finish and publish it before the job fair taking place on 12.10.2022. I intend to publish the full source code of the website for viewing in the future, now I may send it on request if you wish to review it.

Discord Bot

Languages: Python

Software: replit.com

Link

Project info

This project is a bot for the Discord platform. The main purpose of the bot is assisting users that play online tabletop games. The bot has been fine tuned to a modified version of Warhammer Fantasy and supports creating and modifying player characters. Each character is a class with different statistics, traits and an inventory of items.

User-friendly design

Bot allows users to modify the data of their character with the use of simple commands, or dynamically generate a new character in accordance with the game rules. It has been designed to be friendly towards new users and to make often difficult parts of tabletop gaming simple and accessible. Bot also recognizes users and allows them to log into their character, to make managing it quicker.

What I learned

Developing it has helped me understand the user-first approach and the principles of creating a responsive and interactive application.
